# 任务卡模板

复杂任务交给线程前，优先整理成任务卡，减少误解和 token 消耗。用户不需要自己写任务卡，由总控整理。

## 模板

```text
任务：

目标：

负责人：

范围：

不做：

验收：

预览：

是否需要推送：

相关文档：
```

## 示例

```text
任务：重做关卡编辑器界面骨架

目标：
让 level-editor.html 变成清楚可用的开发者工作台。

负责人：
关卡编辑器线程。

范围：
level-editor.html、level-editor.css、level-editor.js。

不做：
不改玩家正式页面，不改玩法规则，不直接发布线上关卡。

验收：
手机可选关、放猪、改方向、删除、导出给总控。

预览：
http://127.0.0.1:8033/level-editor.html

是否需要推送：
完成后交回总控，暂不自行推送。

相关文档：
docs/LEVEL_EDITOR_PLAN.md
```
